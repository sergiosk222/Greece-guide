# Greece Guide — рабочая инструкция

Greece Guide — бесплатный справочник, который объясняет официальные греческие бюрократические процедуры простым человеческим языком.

## Цель проекта

Помочь жителям Греции быстрее понять:

- какие документы нужны;
- куда обращаться;
- какие шаги делать;
- где находится официальный источник;
- когда информация была проверена.

Проект не заменяет официальные сайты и не даёт юридических гарантий.

## Технологии

- Vite
- React
- Motion
- Lucide Icons

## Как добавить статью

1. Открой `src/data/articles.js`.
2. Скопируй объект существующей статьи.
3. Измени:
   - id
   - title
   - category
   - summary
   - status
   - lastChecked
   - sourceName
   - sourceUrl
   - sections
4. Новая статья всегда должна начинаться со статуса:

```js
status: 'draft'
```

5. После ручной проверки владелец проекта может изменить статус на:

```js
status: 'verified'
```

## Правила источников

Использовать только официальные источники:

- gov.gr
- efka.gov.gr
- ypergasias.gov.gr
- aade.gr
- dypa.gov.gr
- eopyy.gov.gr
- migration.gov.gr
- stratologia.gr
- официальные сайты Δήμος

Не использовать как основной источник:

- форумы;
- Facebook;
- Reddit;
- блоги;
- сайты посредников;
- старые новости.

## Статусы статей

- `draft` — черновик, не проверен владельцем.
- `verified` — вручную проверено владельцем.
- `outdated` — нужно обновить.

## Агентные промпты

Папка:

```text
agent-prompts/
```

Содержит инструкции для AI/Codex-агентов:

- research-agent.md
- editor-agent.md
- quality-check-agent.md
- publisher-agent.md

Агенты помогают готовить статьи, но не должны публиковать их без ручного подтверждения владельца.

Информация на сайте объясняет официальные источники простым языком. Перед любым действием нужно проверить актуальность информации на официальном сайте.
