# Greece Guide

Greece Guide — бесплатный справочник, который объясняет греческие официальные бюрократические процедуры простым человеческим языком.

## Цель проекта

Помочь жителям Греции быстро понять:

- какие документы нужны;
- куда обращаться;
- где официальный источник;
- что проверить перед действием;
- какие ошибки чаще всего делают люди.

Проект не заменяет gov.gr, AADE, e-EFKA, DYPA, EOPYY или другие официальные службы. Он только объясняет официальные источники проще.

## Как добавить статью

Открой файл:

```text
src/data/articles.js
```

Добавь новый объект в массив `articles`.

Минимальная структура:

```js
{
  id: 'example-id',
  title: 'Название статьи',
  category: 'Документы',
  summary: 'Краткое описание.',
  status: 'draft',
  lastChecked: '2026-06-13',
  sourceName: 'gov.gr',
  sourceUrl: 'https://www.gov.gr/',
  sections: {
    short: 'Короткое объяснение.',
    whoNeeds: 'Кому нужно.',
    documents: 'Какие документы нужны.',
    where: 'Куда обращаться.',
    steps: ['Шаг 1', 'Шаг 2'],
    mistakes: ['Ошибка 1', 'Ошибка 2'],
    warning: 'Информация объясняет официальный источник простым языком. Перед действием проверьте актуальность на официальном сайте.'
  }
}
```

## Статусы статей

- `draft` — черновик, не подтверждён вручную.
- `verified` — проверено владельцем проекта.
- `outdated` — нужно обновить.

Новые статьи всегда начинаются со статуса `draft`.

## Агентная схема

В папке `agent-prompts` лежат инструкции для помощников:

- `research-agent.md` — поиск официальных источников;
- `translator-agent.md` — перевод бюрократического языка в простой;
- `editor-agent.md` — сборка статьи;
- `quality-check-agent.md` — проверка точности и безопасности;
- `publisher-agent.md` — правила публикации;
- `source-monitor-agent.md` — будущая проверка изменений источников.

Главное правило: агент может готовить черновик, но публикация происходит только после ручной проверки владельцем проекта.

## Контроль качества статьи

Перед сменой статуса на `verified` проверь:

- есть официальный источник;
- есть `sourceUrl`;
- есть `lastChecked`;
- нет выдуманных документов;
- нет обещаний результата;
- текст понятен обычному человеку;
- есть предупреждение проверить официальный сайт.

## Важно

Greece Guide не является юридической консультацией. Перед действиями нужно проверять актуальность информации на официальном сайте.
